ZipVote

Information & Web Design · 2026

Project Objective

To design an intuitive web platform that empowers citizens to easily discover and engage with local electoral information based on their zip code.

Project Overview

ZipVote bridges the gap between complex civic data and everyday users. By prioritizing clear information architecture and accessible user interfaces, the project transforms dense legislative and electoral data into an engaging web experience, increasing local civic awareness and participation.
